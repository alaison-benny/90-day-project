ഇപ്പോൾ നമ്മൾ **Phase 2**-ന്റെ അവസാന ഘട്ടത്തിലാണ്. സർവർ ഓണാക്കി, അത് വർക്ക് ചെയ്യുന്നുണ്ടെന്ന് ഉറപ്പുവരുത്തിയ ശേഷം നമുക്ക് `Dockerfile` ഉണ്ടാക്കി ഇമേജ് ബിൽഡ് ചെയ്യാം.


---

### **Step 1: സർവർ വീണ്ടും ഓണാക്കാം (FastAPI)**
ആദ്യം നമ്മുടെ `venv` ആക്ടിവേറ്റ് ചെയ്ത് ആപ്പ് റൺ ചെയ്യണം.

1. **Terminal-ൽ ഇത് ടൈപ്പ് ചെയ്യുക:**
   ```bash
   source venv/bin/activate
   python app.py
   ```
2. **കാത്തിരിക്കുക:** ടെർമിനലിൽ `INFO: Uvicorn running on http://0.0.0.0:8000` എന്ന് വരുന്നത് വരെ വെയിറ്റ് ചെയ്യുക.

---

### **Step 2: `Dockerfile` നിർമ്മിക്കാം**
സർവർ റൺ ആകുന്നുണ്ടെങ്കിൽ (അത് അവിടെ റൺ ചെയ്തോട്ടെ), **മറ്റൊരു പുതിയ ടെർമിനൽ വിൻഡോ തുറന്ന്** താഴെ പറയുന്നവ ചെയ്യുക:

1. **ഫയൽ ക്രിയേറ്റ് ചെയ്യുക:**
   ```bash
   vim Dockerfile
   ```
2. **താഴെ കാണുന്ന കോഡ് കോപ്പി ചെയ്ത് പേസ്റ്റ് ചെയ്യുക:** (ഇതൊരു പ്രൊഫഷണൽ പ്ലാറ്റ്‌ഫോം എൻജിനീയർ ചെയ്യുന്ന രീതിയിലുള്ള ഫയലാണ്)

```dockerfile
# Base Image
FROM python:3.12-slim

# പ്ലാറ്റ്‌ഫോം എൻജിനീയർ സെറ്റ് ചെയ്യുന്ന വർക്കിംഗ് ഡയറക്ടറി
WORKDIR /app

# സിസ്റ്റം ഡിപെൻഡൻസി ഇൻസ്റ്റാൾ ചെയ്യുന്നു
RUN apt-get update && apt-get install -y curl && rm -rf /var/lib/apt/lists/*

# ആവശ്യമായ ഫയലുകൾ കോപ്പി ചെയ്യുന്നു
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# നമ്മുടെ പ്രൊജക്റ്റ് ഫയലുകളും MLflow ഫോൾഡറും കോപ്പി ചെയ്യുന്നു
COPY . .

# Environment Variables (Hardcoded path മാറ്റാൻ സഹായിക്കുന്നു)
ENV MODEL_PATH="/app/mlruns/1/models/m-553ae89e3eb44e41a87c2cbcca7bc524/artifacts"

# സർവർ പോർട്ട് തുറക്കുന്നു
EXPOSE 8000

# സർവർ സ്റ്റാർട്ട് ചെയ്യാനുള്ള കമാൻഡ്
CMD ["uvicorn", "app:app", "--host", "0.0.0.0", "--port", "8000"]
```



---

### **Step 3: Docker Image ബിൽഡ് ചെയ്യുക**
ഇമേജ് ഉണ്ടാക്കാൻ താഴെ പറയുന്ന കമാൻഡ് അടിക്കുക:
```bash
docker build -t mlops-api:v1 .
```
*(ഇവിടെ `.` മറക്കരുത്. ഇത് നമ്മുടെ നിലവിലുള്ള ഡയറക്ടറിയിൽ നിന്ന് ഇമേജ് ഉണ്ടാക്കാൻ സഹായിക്കും)*

---

### **Step 4: Docker Container റൺ ചെയ്യുക**
ഇമേജ് റെഡിയായാൽ അത് റൺ ചെയ്ത് നോക്കാം:
```bash
docker run -d -p 8000:8000 --name ml-server mlops-api:v1
```
