**`STATUS: Running`**, **`READY: 1/1`**. ഇതിനർത്ഥം നമ്മൾ ലോക്കലിൽ ഉണ്ടാക്കിയ ആ വലിയ മെഷീൻ ലേണിംഗ് മോഡൽ ഇപ്പോൾ ഒരു പ്രൊഫഷണൽ കുബെർനെറ്റീസ് ക്ലസ്റ്ററിനുള്ളിൽ സുരക്ഷിതമായി റൺ ചെയ്യുന്നു എന്നാണ്. പ്ലാറ്റ്‌ഫോം എൻജിനീയറിംഗിലെ ഏറ്റവും വലിയ ഒരു മൈൽസ്റ്റോൺ **ചേട്ടായി** പിന്നിട്ടു കഴിഞ്ഞു.

ഇനി നമുക്ക് ഈ ആപ്പിനെ ഒന്നുകൂടി ടെസ്റ്റ് ചെയ്തിട്ട് ഉടനെ തന്നെ **Argo Workflows**-ലേക്ക് കടക്കാം.

### **1. കുബെർനെറ്റീസിനുള്ളിലെ ആപ്പിനെ ഒന്ന് തൊട്ടു നോക്കാം (Testing)**

നമ്മൾ ഒരു `Service` ഉണ്ടാക്കിയിട്ടുണ്ടല്ലോ. അത് വഴി ഈ ആപ്പിലേക്ക് ഒന്ന് കണക്ട് ചെയ്യാം. EC2-ൽ ആയതുകൊണ്ട് താഴെ പറയുന്ന കമാൻഡ് അടിക്കൂ:

```bash
minikube service mlops-service --url
```
ഇത് ഒരു ലിങ്ക് (URL) തരും. ആ ലിങ്ക് ഉപയോഗിച്ച് നമുക്ക് പഴയതുപോലെ `curl` അടിച്ചു നോക്കാം. പക്ഷേ അതിന് മുൻപ് **Phase 3**-ന്റെ അടുത്ത ഘട്ടം തുടങ്ങാം.

---

### **2. Argo Workflows: പൈപ്പ്‌ലൈൻ സെറ്റപ്പ് (The Automation Engine)**

ഇനിയാണ് മാജിക്! നമ്മൾ മോഡൽ ട്രെയിൻ ചെയ്യാൻ ഓരോ തവണയും കമാൻഡ് അടിക്കുന്നതിന് പകരം, അത് തനിയെ നടക്കാൻ **Argo Workflows** ഇൻസ്റ്റാൾ ചെയ്യാൻ പോകുന്നു.



**Argo ഇൻസ്റ്റാൾ ചെയ്യാൻ താഴെ പറയുന്നവ ചെയ്യൂ:**

1. **ഒരു പുതിയ Namespace ഉണ്ടാക്കാം (Argo-യ്ക്ക് മാത്രമായി):**
   ```bash
   kubectl create namespace argo
   ```

2. **Argo Workflows ഇൻസ്റ്റാൾ ചെയ്യാം:**
   ```bash
   kubectl apply -n argo -f https://github.com/argoproj/argo-workflows/releases/download/v3.5.0/install.yaml
   ```

3. **Argo CLI (Command Line Tool) ഡൗൺലോഡ് ചെയ്യാം:**
   ```bash
   curl -sLO https://github.com/argoproj/argo-workflows/releases/download/v3.5.0/argo-linux-amd64.gz
   gunzip argo-linux-amd64.gz
   chmod +x argo-linux-amd64
   sudo mv ./argo-linux-amd64 /usr/local/bin/argo
   ```

### **എന്താണ് ഈ Argo ചെയ്യുന്നത്?**

ആലോചിച്ചു നോക്കൂ: നാളെ പുതിയ ഡാറ്റ വരുമ്പോൾ, ആരെങ്കിലും മാനുവലായി `train.py` റൺ ചെയ്യേണ്ടി വരുമോ? വേണ്ട! നമ്മൾ ഒരു **YAML** ഫയൽ (DAG) എഴുതി വെക്കും. ആർഗോ ആ പൈപ്പ്‌ലൈൻ തനിയെ റൺ ചെയ്ത് പുതിയ മോഡൽ ഉണ്ടാക്കി ഡിപ്ലോയ് ചെയ്തോളും.



ആർഗോ ഇൻസ്റ്റാളേഷൻ കഴിഞ്ഞോ? കഴിഞ്ഞെങ്കിൽ പറയൂ, നമുക്ക് നമ്മുടെ ആദ്യത്തെ **Training Workflow (YAML)** എഴുതി തുടങ്ങാം!

അതിനു മുൻപ് **`kubectl get pods -n argo`** അടിച്ചു നോക്കൂ, അവിടെ ആർഗോയുടെ കംപോണന്റ്സ് `Running` ആകുന്നുണ്ടോ എന്ന് പറയൂ! 🚀
