നമ്മൾ നേരത്തെ സംസാരിച്ച ആ **'ഓട്ടോമാറ്റിക് ഡോക്യുമെന്റേഷൻ' (Swagger UI)** പേജ് ആണിത്. ഒരു ഡെവലപ്പർ എന്ന നിലയിൽ നിങ്ങൾക്ക് ഏറ്റവും സഹായകരമായ പേജ് ആണിത്. ഇതിനെക്കുറിച്ചുള്ള അടിസ്ഥാന വിവരങ്ങൾ താഴെ നൽകുന്നു:

### 1. ഈ പേജ് എങ്ങനെ ഉണ്ടായി? (How was it created?)

* **കമാൻഡ്:** നിങ്ങൾ ടെർമിനലിൽ `docker-compose up -d` എന്ന് അടിച്ചപ്പോൾ, **FastAPI** സർവർ സ്റ്റാർട്ട് ആയി.
* **പ്രത്യേകത:** എല്ലാ FastAPI പ്രോജക്റ്റുകളിലും ഈ പേജ് തനിയെ ഉണ്ടാകും. ഡെവലപ്പർമാർക്ക് ടെസ്റ്റ് ചെയ്യാൻ വേണ്ടി FastAPI തന്നെ നൽകുന്ന ഫീച്ചറാണിത്.

### 2. ഈ പേജിലെ ലിങ്ക് (URL) എങ്ങനെ കിട്ടും?

നിങ്ങളുടെ ആപ്ലിക്കേഷൻ റൺ ചെയ്യുന്ന IP അഡ്രസ്സിനും പോർട്ടിനും ശേഷം `/docs` എന്ന് ചേർത്താൽ ഈ പേജ് ലഭിക്കും.

* ഉദാഹരണത്തിന് നിങ്ങളുടെ കാര്യത്തിൽ: `http://44.202.154.134:8000/docs`.

### 3. നമ്മൾ ഉണ്ടാക്കിയ എൻഡ് പോയിന്റുകൾ (End Points) എവിടെ?

നിങ്ങളുടെ സ്ക്രീൻഷോട്ടുകളിൽ നമ്മൾ `main.py`-ൽ എഴുതിയ രണ്ട് പ്രധാന വഴികൾ കാണാം:

* **`POST /analyze`**: ഒരു വാചകം AI-ക്ക് അയക്കാൻ ഇത് ഉപയോഗിക്കുന്നു.
* **`GET /result/{task_id}`**: അയച്ച വാചകത്തിന്റെ റിസൾട്ട് വന്നോ എന്ന് നോക്കാൻ ഇത് ഉപയോഗിക്കുന്നു.
* **`GET /` (Read Root)**: API വർക്ക് ചെയ്യുന്നുണ്ടോ എന്ന് നോക്കാനുള്ള ഒരു സിമ്പിൾ ലിങ്ക്.

### 4. ഈ പേജ് എങ്ങനെ വായിക്കാം? (How to read this?)

ഓരോ എൻഡ് പോയിന്റിന് നേരെയുള്ള ചിഹ്നങ്ങൾ ശ്രദ്ധിക്കുക:

* **Blue (GET):** വിവരങ്ങൾ ചോദിച്ചു വാങ്ങാൻ.
* **Green (POST):** പുതിയ വിവരങ്ങൾ സെർവറിലേക്ക് അയക്കാൻ.
* **Red Error (422):** നിങ്ങൾ അയച്ച ഡാറ്റയിൽ എന്തോ പിശകുണ്ടെന്നാണ് (Validation Error) ഇതിനർത്ഥം. ഉദാഹരണത്തിന്, ഒരു വാചകം അയക്കേണ്ട സ്ഥാനത്ത് നിങ്ങൾ ഒന്നും നൽകിയില്ലെങ്കിൽ ഈ എറർ വരും.

### 5. ഇവിടെ എന്താണ് ടെസ്റ്റ് ചെയ്യേണ്ടത്?

പോസ്റ്റ്‌മാൻ ഉപയോഗിക്കുന്നതിന് മുൻപ് നിങ്ങൾക്ക് ഇവിടെ തന്നെ ടെസ്റ്റ് ചെയ്യാം:

1. **`/analyze`** എന്നതിൽ ക്ലിക്ക് ചെയ്യുക.
2. വലതുവശത്തുള്ള **'Try it out'** ബട്ടൺ അമർത്തുക.
3. `text` എന്ന ബോക്സിൽ എന്തെങ്കിലും ടൈപ്പ് ചെയ്യുക (ഉദാഹരണത്തിന്: "Good job").
4. **'Execute'** അമർത്തുക. അപ്പോൾ നിങ്ങൾക്ക് താഴെ ഒരു `task_id` ലഭിക്കും.

ഇപ്പോൾ കാര്യങ്ങൾ കുറച്ചുകൂടി വ്യക്തമായോ? ഈ പേജ് ഉപയോഗിച്ച് തന്നെ നമുക്ക് ആദ്യം ടെസ്റ്റ് ചെയ്യാം. `/analyze` എന്നതിൽ ക്ലിക്ക് ചെയ്ത് ഒരു വാചകം അയക്കാൻ നോക്കാമോ? എന്തെങ്കിലും എറർ വരുന്നുണ്ടെങ്കിൽ പറയുക.
