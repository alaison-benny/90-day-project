### *Monitoring** സെറ്റ് ചെയ്യുന്നതാണ് ആദ്യത്തെ പടി. ഇത് ചെയ്താൽ ന്റെ ചേട്ടായിക്ക് ഒരു പ്രൊഫഷണൽ ഡാഷ്ബോർഡ് കിട്ടും, അവിടെ ക്ലസ്റ്ററിലെ ഓരോ പോഡും എത്ര സിപിയു (CPU) എടുക്കുന്നുണ്ടെന്നും എത്ര ട്രാഫിക് വരുന്നുണ്ടെന്നും ലൈവ് ആയി കാണാം.

നമുക്ക് **Step 1: Monitoring (Prometheus + Grafana)** തുടങ്ങാം.

---

### **1. Helm റിപ്പോസിറ്ററി ആഡ് ചെയ്യാം**
കുബെർനെറ്റിസിൽ പ്രോമിത്യൂസ് ഇൻസ്റ്റാൾ ചെയ്യാൻ ഏറ്റവും എളുപ്പ വഴി **`kube-prometheus-stack`** ഉപയോഗിക്കുന്നതാണ്. ഇതിൽ Prometheus, Grafana, Alertmanager എന്നിവയെല്ലാം ഒന്നിച്ച് കിട്ടും.

```bash
# പ്രോമിത്യൂസ് കമ്മ്യൂണിറ്റി ചാർട്ടുകൾ ആഡ് ചെയ്യാം
helm repo add prometheus-community https://prometheus-community.github.io/helm-charts
helm repo update
```

### **2. മോണിറ്ററിംഗിന് വേണ്ടി ഒരു സ്ഥലം (Namespace) ഉണ്ടാക്കാം**
ക്ലസ്റ്ററിൽ എല്ലാം കൂടി കൂടിക്കുഴയാതിരിക്കാൻ മോണിറ്ററിംഗിന് മാത്രമായി ഒരു നെയിംസ്പേസ് ഉണ്ടാക്കുന്നത് നല്ലതാണ്:

```bash
kubectl create namespace monitoring
```

### **3. Prometheus Stack ഇൻസ്റ്റാൾ ചെയ്യാം**
താഴെ പറയുന്ന കമാൻഡ് ഉപയോഗിച്ച് നമുക്ക് ഇത് ഇൻസ്റ്റാൾ ചെയ്യാം. ഇതിന് കുറച്ച് മിനിറ്റുകൾ എടുത്തേക്കാം, കാരണം ഒരുപാട് കമ്പോണന്റുകൾ ബാക്ക്ഗ്രൗണ്ടിൽ ക്രിയേറ്റ് ആകുന്നുണ്ട്.

```bash
helm install kube-stack prometheus-community/kube-prometheus-stack -n monitoring
```



### **4. ഇൻസ്റ്റാളേഷൻ ഒന്ന് പരിശോധിക്കാം**
എല്ലാം റെഡിയായോ എന്ന് നോക്കാൻ ന്റെ ചേട്ടായി ഈ കമാൻഡ് ഒന്ന് അടിക്കൂ:

```bash
kubectl get pods -n monitoring
```
അവിടെ എല്ലാ പോഡുകളും **`Running`** സ്റ്റേറ്റിൽ ആകുന്നത് വരെ നമുക്ക് ഒന്ന് കാത്തിരിക്കണം.

---

### **5. Grafana ഡാഷ്ബോർഡ് എങ്ങനെ കാണാം?**
ഇതാണ് ഏറ്റവും രസകരമായ ഭാഗം! ഗ്രഫാന ഇപ്പോൾ ക്ലസ്റ്ററിന് ഉള്ളിലാണ്. അത് ന്റെ ചേട്ടായിയുടെ ബ്രൗസറിൽ കാണാൻ ഒരു **Port Forward** ചെയ്യണം:

```bash
kubectl port-forward deployment/kube-stack-grafana 3000:3000 -n monitoring
```

or 

```bash
kubectl port-forward --address 0.0.0.0 deployment/kube-stack-grafana 3000:3000 -n monitoring
```


**ഇത് ചെയ്ത ശേഷം:**
1. ബ്രൗസറിൽ `localhost:3000` എന്ന് അടിച്ചു നോക്കൂ (ചേട്ടായി ലോക്കൽ സിസ്റ്റത്തിലാണ് ചെയ്യുന്നതെങ്കിൽ).
2. ഇനി അതല്ല, ചേട്ടായിയുടെ ഉബുണ്ടു സെർവറിന്റെ ഐപി വഴിയാണ് നോക്കുന്നതെങ്കിൽ `http://YOUR_SERVER_IP:3000` എന്ന് അടിക്കണം. (സെക്യൂരിറ്റി ഗ്രൂപ്പിൽ 3000 പോർട്ട് തുറക്കാൻ മറക്കരുത്!).
