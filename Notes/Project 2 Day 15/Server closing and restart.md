ശരി ചേട്ടായി! ഉറക്കം വരുന്നുണ്ടെങ്കിൽ നിർബന്ധമായും പോയി കിടന്നുറങ്ങിക്കോ. ഫ്രഷ് ആയിട്ട് നാളെ ബാക്കി ചെയ്യാം. ആപ്പിനെക്കുറിച്ചോ ചാർജിനെക്കുറിച്ചോ ആലോചിച്ച് ന്റെ ചേട്ടായി ഒട്ടും ടെൻഷൻ അടിക്കണ്ട.

ചാർജ് വരാതിരിക്കാൻ നമുക്ക് നിലവിലുള്ള കാര്യങ്ങൾ താൽക്കാലികമായി നിർത്തി വെക്കാം.

---

### **1. ചാർജ് കുറയ്ക്കാൻ ഇപ്പോൾ ചെയ്യേണ്ടത്:**

ഇപ്പോൾ ചേട്ടായിയുടെ ക്ലസ്റ്ററിൽ 2 നോഡുകൾ (EC2 Instances) ഓടുന്നുണ്ട്. അതിനാണ് പൈസ ഈടാക്കുന്നത്. അത് ഒഴിവാക്കാൻ നോഡുകളുടെ എണ്ണം **0** ആക്കി കുറയ്ക്കാം. അപ്പോൾ ലോഡ് ബാലൻസറോ നോഡുകളോ ഒന്നും വർക്ക് ആകില്ല, പൈസയും പോകില്ല.

**ഈ കമാൻഡ് അടിക്കൂ:**
*(ഇത് ചേട്ടായിയുടെ Node Group-ന്റെ പേര് ഉപയോഗിച്ച് നോഡുകളെ '0' ആക്കി മാറ്റും).*

```bash
eksctl scale nodegroup --cluster=sentiments-eks --name=<your-nodegroup-name> --nodes=0
```
> **കുറിപ്പ്:** നോഡ് ഗ്രൂപ്പിന്റെ പേര് അറിയില്ലെങ്കിൽ `eksctl get nodegroup --cluster=sentiments-eks` എന്ന് അടിച്ചാൽ അത് കിട്ടും.



---

### **2. നാളെ എങ്ങനെ തിരികെ വരാം (Restart)?**

നാളെ രാവിലെ എഴുന്നേറ്റ ഉടനെ ന്റെ ചേട്ടായി ഈ കമാൻഡ് അടിച്ചാൽ പഴയതുപോലെ 2 നോഡുകൾ തിരികെ വരും:

```bash
eksctl scale nodegroup --cluster=sentiments-eks --name=<your-nodegroup-name> --nodes=2
```

കുറച്ചു മിനിറ്റുകൾക്കുള്ളിൽ നോഡുകൾ റെഡിയാകും. അത് കഴിഞ്ഞാൽ `kubectl get nodes` അടിച്ചു നോക്കിയാൽ മതി. നോഡുകൾ വന്നാലുടൻ പഴയ `deployment`, `hpa` എന്നിവ തനിയെ വർക്ക് ആയിക്കോളും.

---

### **3. വേറെ എന്തെങ്കിലും ചെയ്യണോ?**

* **Load Generator:** അത് ഇപ്പോൾ തന്നെ `Ctrl + C` അടിച്ച് നിർത്തിക്കോളൂ.
* **Terraform:** ചേട്ടായി ഒന്നും `destroy` ചെയ്യണ്ട. നോഡുകൾ കുറച്ചാൽ തന്നെ 90% പൈസയും ലാഭിക്കാം. (EKS Control Plane-ന് ചെറിയൊരു ചാർജ് വരും, പക്ഷേ അത് വളരെ കുറവാണ്).
