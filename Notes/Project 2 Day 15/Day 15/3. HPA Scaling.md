സൂപ്പർ ചേട്ടായി! രണ്ട് പോഡുകളും റെഡിയായിക്കഴിഞ്ഞു. ഇനി നമുക്ക് ഇവയെ ഓട്ടോമാറ്റിക്കായി സ്കെയിൽ (Scale) ചെയ്യാൻ പഠിക്കാം. 

ഇതിനായി ആദ്യം നമുക്ക് **Metrics Server** ആവശ്യമാണ്. ക്ലസ്റ്ററിലെ പോഡുകൾ എത്രമാത്രം CPU-ഉം മെമ്മറിയും ഉപയോഗിക്കുന്നു എന്ന് അളന്ന് എച്ച്.പി.എ-യ്ക്ക് (HPA) വിവരം നൽകുന്നത് ഇയാളാണ്.

---

### **Step 1: Metrics Server ഇൻസ്റ്റാൾ ചെയ്യാം**

EKS-ൽ Metrics Server നേരിട്ട് ഉണ്ടാകില്ല. താഴെ പറയുന്ന ഒരൊറ്റ കമാൻഡ് ഉപയോഗിച്ച് ന്റെ ചേട്ടായിക്ക് ഇത് ഇൻസ്റ്റാൾ ചെയ്യാം:

```bash
kubectl apply -f https://github.com/kubernetes-sigs/metrics-server/releases/latest/download/components.yaml
```

ഇൻസ്റ്റാൾ ആയോ എന്ന് നോക്കാൻ ഈ കമാൻഡ് അടിക്കൂ:
```bash
kubectl get deployment metrics-server -n kube-system
```



---

### **Step 2: Horizontal Pod Autoscaler (HPA) സെറ്റ് ചെയ്യാം**

ഇനി നമുക്ക് നമ്മുടെ ആപ്പിന് വേണ്ടി ഒരു സ്കെയിലിംഗ് റൂൾ ഉണ്ടാക്കാം. നമ്മുടെ ആപ്പിന്റെ CPU ഉപയോഗം **50%**-ൽ കൂടുതൽ പോയാൽ, 2 പോഡുകൾ ഉള്ളത് **5** എണ്ണം വരെയായി വർദ്ധിപ്പിക്കണം എന്ന് നമുക്ക് പറയാം.

ഇതിനായി `hpa.yaml` എന്നൊരു ഫയൽ ഉണ്ടാക്കി താഴെ പറയുന്നവ ചേർക്കുക:

```yaml
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: sentiment-api-hpa
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: sentiment-api
  minReplicas: 2
  maxReplicas: 5
  metrics:
  - type: Resource
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 50
```

ഇത് അപ്ലൈ ചെയ്യാൻ:
```bash
kubectl apply -f k8s/hpa.yaml
```

---

### **Step 3: സ്കെയിലിംഗ് ചെക്ക് ചെയ്യാം**

ഇപ്പോൾ എല്ലാം സെറ്റ് ആയിക്കഴിഞ്ഞു. സ്റ്റാറ്റസ് നോക്കാൻ താഴെ പറയുന്ന കമാൻഡ് അടിക്കൂ:

```bash
kubectl get hpa
```



**ശ്രദ്ധിക്കുക:** തുടക്കത്തിൽ `TARGETS` എന്ന കോളത്തിൽ `<unknown>/50%` എന്ന് കാണിച്ചേക്കാം. ഒരു മിനിറ്റ് കഴിഞ്ഞാൽ അത് `0%/50%` എന്നായി മാറും. 

---

### **അടുത്ത പടി: ലോഡ് ടെസ്റ്റിംഗ് (The Fun Part!)**

ന്റെ ചേട്ടായിക്ക് ഇപ്പോൾ ഒരു സംശയം ഉണ്ടാകും, *"ലോഡ് കൂടുമ്പോൾ ഇത് ശരിക്കും വർക്ക് ആകുമോ?"* എന്ന്. അത് പരീക്ഷിക്കാൻ നമുക്ക് ഒരു **Load Generator** ഓടിക്കാം. 

**ന്റെ ചേട്ടായി റെഡിയാണോ?** നമുക്ക് കൃത്രിമമായി ലോഡ് ഉണ്ടാക്കി 2 പോഡുകൾ 5 ആയി മാറുന്നത് നേരിട്ട് കാണാം! 

**തുടങ്ങട്ടേ ചേട്ടായി?** അതോ ഇതിൽ എന്തെങ്കിലും സംശയം ഉണ്ടോ?
