### ഈ ഡയഗ്രം മൂന്ന് പ്രധാന ഭാഗങ്ങളായി തിരിക്കാം:

1. ഡെവലപ്പർ ഫ്ലോ (ഇടതുവശത്ത്):

ന്റെ ചേട്ടായി FastAPI ഉപയോഗിച്ച് മോഡൽ ഉണ്ടാക്കുന്നു.

അത് Docker ഉപയോഗിച്ച് കണ്ടെയ്നറൈസ് ചെയ്യുന്നു.

കോഡ് GitHub-ലേക്ക് പുഷ് ചെയ്യുന്നു.

(ഇനി നമ്മൾ ചെയ്യാൻ പോകുന്ന GitHub Actions വഴിയുള്ള സിഐ/സിഡി ഫ്ലോയും ഇതിൽ കാണിച്ചിട്ടുണ്ട്).

2. ഇൻഫ്രാസ്ട്രക്ചർ & ഡിപ്ലോയ്മെന്റ് (വലതുവശത്ത്):

Terraform ഉപയോഗിച്ചാണ് EKS ക്ലസ്റ്റർ സെറ്റ് ചെയ്യുന്നത്.

Helm Charts, Kubernetes Manifests എന്നിവ വഴിയാണ് ഡിപ്ലോയ്മെന്റ് മാനേജ് ചെയ്യുന്നത്.

3. AWS Cloud (നടുവിൽ):

ഇതാണ് പ്രധാന ഭാഗം. ഒരു VPC-ക്കുള്ളിൽ EKS Cluster ഓടുന്നത് കാണാം.

External Traffic (HTTP/S) നേരെ AWS ALB വഴി Ingress Controller-ൽ എത്തുന്നു.

അവിടെ നിന്ന് അത് Sentiment API Pods-ലേക്ക് പോകുന്നു.

Auto-Scaling (HPA), API Pods-നെ കൺട്രോൾ ചെയ്യുന്നത് കാണിച്ചിട്ടുണ്ട്.

Prometheus Operator + Grafana വഴി പോഡുകളിലെ വിവരങ്ങൾ Scrape ചെയ്ത് മോണിറ്റർ ചെയ്യുന്നു.

IAM Roles, Secrets Manager എന്നിവയുടെ കണക്ഷനും ഇതിൽ വ്യക്തമാണ്.
