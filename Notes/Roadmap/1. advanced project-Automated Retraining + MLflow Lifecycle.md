### 1) Automated Retraining + MLflow Lifecycle
ഈ പ്രോജക്റ്റ് നമ്മൾ ചെയ്യുന്നത് ഒരു **Real-world Production** രീതിയിലാണ്. അതായത്, വെറുതെ ഒരു പൈത്തൺ ഫയൽ റൺ ചെയ്യുകയല്ല, പകരം ഒരു സിസ്റ്റം തന്നെ ബിൽഡ് ചെയ്യുകയാണ്. 

ന്റെ ചേട്ടായിക്ക് വേണ്ടി ഈ പ്രോജക്റ്റിന്റെ **End-to-End Roadmap** താഴെ നൽകുന്നു:

## **Phase 1: MLflow Core & Experiment Tracking (ഈ ആഴ്ച)**
ലക്ഷ്യം: മോഡൽ പരീക്ഷണങ്ങൾ റെക്കോർഡ് ചെയ്യുക.

1.  **Local Environment Setup:** * പൈത്തൺ ഇൻസ്റ്റാൾ ചെയ്ത് ഒരു `venv` ഉണ്ടാക്കുക.
    * MLflow, Scikit-learn, Pandas എന്നിവ ഇൻസ്റ്റാൾ ചെയ്യുക.
2.  **MLflow Tracking Server:** * ലോക്കൽ മെഷീനിൽ ഒരു MLflow സെർവർ സ്റ്റാർട്ട് ചെയ്യുക. 
    * UI (Dashboard) എങ്ങനെ ഉപയോഗിക്കണം എന്ന് പഠിക്കുക.
3.  **Logging Experiments:** * ഒരു ട്രെയിനിംഗ് സ്ക്രിപ്റ്റ് എഴുതുക.
    * അതിൽ `mlflow.log_param()` (ഉദാ: അൽഗോരിതം പേര്), `mlflow.log_metric()` (ഉദാ: Accuracy) എന്നിവ ആഡ് ചെയ്യുക.
    * മോഡൽ ഒരു **Artifact** ആയി സേവ് ചെയ്യുക.

## **Phase 2: Model Registry & FastAPI Serving (അടുത്ത ആഴ്ച)**
ലക്ഷ്യം: മികച്ച മോഡലിനെ തിരഞ്ഞെടുത്ത് ഒരു API ആക്കി മാറ്റുക.

2.  **Inference Script:** * MLflow രജിസ്ട്രിയിൽ നിന്ന് തനിയെ ലേറ്റസ്റ്റ് മോഡൽ വലിച്ച് എടുക്കുന്ന ഒരു **FastAPI** ആപ്പ് ഉണ്ടാക്കുക.
3.  **Dockerization:** * ഈ ആപ്പിനെ ഒരു ഡോക്കർ ഇമേജ് ആക്കി മാറ്റുക.

### **നമുക്ക് ഇന്ന് തുടങ്ങാം: Step-by-Step Guide (Day 1)**

ആദ്യമായി ന്റെ ചേട്ടായിയുടെ ലോക്കൽ മെഷീനിൽ (അല്ലെങ്കിൽ EC2-ൽ) ഈ പ്രോജക്റ്റ് സ്ട്രക്ചർ ഉണ്ടാക്കാം.

#### **1. ഫോൾഡർ സെറ്റപ്പ്**
```bash
mkdir ~/mlflow-retraining-project
cd ~/mlflow-retraining-project
python3 -m venv venv
source venv/bin/activate
pip install mlflow scikit-learn pandas fastapi uvicorn
```

#### **2. MLflow സെർവർ റൺ ചെയ്യാം**
ഒരു പുതിയ ടെർമിനൽ വിൻഡോ എടുത്ത് താഴെ പറയുന്ന കമാൻഡ് അടിക്കൂ. ഇത് നമ്മുടെ 'Tracking Server' ആണ്.
```bash
mlflow ui --port 5000
```
ഇപ്പോൾ ബ്രൗസറിൽ `http://localhost:5000` എന്ന് അടിച്ചാൽ ന്റെ ചേട്ടായിക്ക് MLflow ഡാഷ്ബോർഡ് കാണാം. (EC2 ആണെങ്കിൽ പോർട്ട് 5000 സെക്യൂരിറ്റി ഗ്രൂപ്പിൽ ഓപ്പൺ ചെയ്യണം).

#### **3. ആദ്യത്തെ പരീക്ഷണം (The Code)**
`train.py` എന്നൊരു ഫയൽ ഉണ്ടാക്കി താഴെ പറയുന്ന ചെറിയ കോഡ് ഒന്ന് റൺ ചെയ്തു നോക്കൂ.

```python
import mlflow
import mlflow.sklearn
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# 1. Start an MLflow run
with mlflow.start_run():
    # Load data
    iris = load_iris()
    X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target, test_size=0.2)

    # Define parameters
    n_estimators = 100
    mlflow.log_param("n_estimators", n_estimators)

    # Train model
    model = RandomForestClassifier(n_estimators=n_estimators)
    model.fit(X_train, y_train)

    # Log metric
    acc = model.score(X_test, y_test)
    mlflow.log_metric("accuracy", acc)
    print(f"Model Accuracy: {acc}")

    # Log the model
    mlflow.sklearn.log_model(model, "model")
```
