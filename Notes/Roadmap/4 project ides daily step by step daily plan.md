## 📌 Project 1: Model Serving API (FastAPI + Docker + AWS)  
**Duration: 14 days**

**Day 1–2:**  
- Python basics refresh (virtualenv, requirements.txt)  
- Simple ML model (sentiment analysis with scikit-learn)  

**Day 3–4:**  
- FastAPI basics (routes, request/response models)  
- Integrate ML model into FastAPI endpoint  

**Day 5–6:**  
- Docker basics (Dockerfile, docker-compose)  
- Containerize FastAPI app  

**Day 7–8:**  
- AWS EC2 setup (security groups, SSH, keypairs)  
- Deploy Dockerized app to EC2  

**Day 9–10:**  
- Add logging (Python logging, AWS CloudWatch basics)  
- Test API with Postman  

**Day 11–12:**  
- Monitoring setup (Prometheus scrape config, Grafana dashboard)  

**Day 13–14:**  
- Documentation (README, architecture diagram)  
- Push repo to GitHub

**Day 19–21:**

- Documentation + GitHub repo update
