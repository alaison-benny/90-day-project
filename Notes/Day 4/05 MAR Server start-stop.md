### **1. സെർവർ എങ്ങനെ ഓഫ് ചെയ്യാം?**

നിങ്ങൾക്ക് രണ്ട് വഴികളുണ്ട്:

* **Terraform വഴി (ശുപാർശ ചെയ്യുന്നത്):** നിങ്ങൾ ടെർമിനലിൽ `terraform destroy` എന്ന് അടിച്ചാൽ AWS-ൽ ഉള്ള സെർവറും സെക്യൂരിറ്റി ഗ്രൂപ്പുകളും എല്ലാം ഡിലീറ്റ് ആകും. ഇത് പൈസ ലാഭിക്കാൻ നല്ലതാണ്. പക്ഷേ, നാളെ വരുമ്പോൾ വീണ്ടും Kafka-യും `venv`-ഉം ഒക്കെ ഇൻസ്റ്റാൾ ചെയ്യേണ്ടി വരും.
* **AWS Console വഴി (സൗകര്യപ്രദം):** AWS കൺസോളിൽ പോയി നിങ്ങളുടെ EC2 ഇൻസ്റ്റൻസ് **"Stop"** ചെയ്യുക. ഇത് ചെയ്താൽ നിങ്ങളുടെ ഫയലുകളും സെറ്റിംഗ്‌സും അവിടെത്തന്നെ ഉണ്ടാകും. നാളെ "Start" ചെയ്താൽ മാത്രം മതി.

* Path: ~/Langchain-Chatchat/docker/ai_worker_project

### **2. നാളെ എങ്ങനെ തിരികെ വരാം?**

നാളെ സെർവർ ഓൺ ചെയ്ത ശേഷം താഴെ പറയുന്ന സ്റ്റെപ്പുകൾ ചെയ്താൽ നമ്മൾ നിർത്തിയ ഇടത്തുനിന്നും തുടങ്ങാം:

1. **IP മാറ്റം ശ്രദ്ധിക്കുക:** സെർവർ ഓഫ് ചെയ്ത് ഓൺ ചെയ്യുമ്പോൾ IP അഡ്രസ്സ് മാറിയേക്കാം. MobaXterm-ൽ പുതിയ IP നൽകി ലോഗിൻ ചെയ്യുക.
2. **Kafka സ്റ്റാർട്ട് ചെയ്യുക:** നിങ്ങളുടെ `ai_worker_project` ഫോൾഡറിൽ പോയി പഴയതുപോലെ ഡോക്കർ റൺ ചെയ്യുക:
```bash
docker-compose up -d

```


3. **Virtual Environment ആക്ടിവേറ്റ് ചെയ്യുക:**
```bash
source venv/bin/activate

```


4. **AI Worker റൺ ചെയ്യുക:**
```bash
python3 worker.py

```
