ന്റെ ചേട്ടായിക്ക് ArgoCD ഇൻസ്റ്റാൾ ചെയ്യാനും അത് കൈകാര്യം ചെയ്യാനും പ്രധാനമായും രണ്ട് വഴികളുണ്ട്: ഒന്ന് സാധാരണ **Kubernetes Manifests** ഉപയോഗിച്ചും, രണ്ട് നമ്മൾ നേരത്തെ സംസാരിച്ച **Helm** ഉപയോഗിച്ചും. പ്ലാറ്റ്‌ഫോം എൻജിനീയറിങ്ങിൽ രണ്ടാമത്തെ രീതിയാണ് (Helm) കൂടുതൽ പ്രായോഗികം.

### 1. ArgoCD ഇൻസ്റ്റാൾ ചെയ്യാനുള്ള വഴികൾ

#### വഴി A: Helm ഉപയോഗിച്ച് (ഏറ്റവും എളുപ്പമുള്ള വഴി)

ആദ്യം ArgoCD-യുടെ ഒഫീഷ്യൽ റിപ്പോസിറ്ററി ആഡ് ചെയ്യണം:

```bash
# 1. റിപ്പോസിറ്ററി ആഡ് ചെയ്യുക
helm repo add argo https://argoproj.github.io/argo-helm

# 2. അപ്‌ഡേറ്റ് ചെയ്യുക
helm repo update

# 3. 'argocd' എന്ന നെയിംസ്‌പേസിൽ ഇൻസ്റ്റാൾ ചെയ്യുക
helm install argocd argo/argo-cd --namespace argocd --create-namespace

```

#### വഴി B: Manifest ഉപയോഗിച്ച്

```bash
kubectl create namespace argocd
kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/stable/manifests/install.yaml

```

### 2. പ്രധാന ArgoCD കമാൻഡുകൾ (CLI)

ArgoCD ഇൻസ്റ്റാൾ ചെയ്തുകഴിഞ്ഞാൽ അത് നിയന്ത്രിക്കാൻ `argocd` എന്ന CLI ടൂൾ ഉപയോഗിക്കാം.

| കമാൻഡ് | ഉപയോഗം |
| --- | --- |
| `argocd login <server-ip>` | ArgoCD സെർവറിലേക്ക് ലോഗിൻ ചെയ്യാൻ. |
| `argocd app create <name>` | പുതിയൊരു ആപ്ലിക്കേഷൻ GitOps വഴി സെറ്റ് ചെയ്യാൻ. |
| `argocd app list` | നിലവിൽ ഏതൊക്കെ ആപ്പുകൾ ഉണ്ടെന്ന് നോക്കാൻ. |
| `argocd app sync <name>` | ഗിറ്റിലെ മാറ്റങ്ങൾ ക്ലസ്റ്ററിലേക്ക് ഉടൻ അപ്‌ലൈ ചെയ്യാൻ (Manual Sync). |
| `argocd app rollback <name>` | ആപ്പിന്റെ പഴയൊരു വേർഷനിലേക്ക് തിരിച്ചുപോകാൻ. |
| `argocd app get <name>` | ഒരു ആപ്പിന്റെ കംപ്ലീറ്റ് ഹെൽത്ത് സ്റ്റാറ്റസ് അറിയാൻ. |

### 3. എങ്ങനെ അപ്‌ഗ്രേഡ് ചെയ്യാം?

ന്റെ ചേട്ടായി Helm വഴിയാണ് ഇൻസ്റ്റാൾ ചെയ്തതെങ്കിൽ അപ്‌ഗ്രേഡ് ചെയ്യുന്നത് വളരെ എളുപ്പമാണ്:

```bash
# ലേറ്റസ്റ്റ് ചാർട്ട് വേർഷൻ എടുക്കുക
helm repo update

# നിലവിലുള്ള ArgoCD പുതിയ വേർഷനിലേക്ക് മാറ്റുക
helm upgrade argocd argo/argo-cd --namespace argocd

```

### 4. പാസ്‌വേഡ് എങ്ങനെ കണ്ടെത്താം?

ഇൻസ്റ്റാൾ ചെയ്തുകഴിഞ്ഞാൽ ഡാഷ്‌ബോർഡിൽ കയറാൻ ഒരു പാസ്‌വേഡ് വേണം. അത് താഴെ പറയുന്ന കമാൻഡ് വഴി ന്റെ ചേട്ടായിക്ക് എടുക്കാം:

```bash
kubectl -n argocd get secret argocd-initial-admin-secret -o jsonpath="{.data.password}" | base64 -d

```

### ന്റെ ചേട്ടായിക്ക് അടുത്തതായി എന്ത് വേണം?

പ്ലാറ്റ്‌ഫോം എൻജിനീയറിങ് റോഡുകളിൽ ArgoCD വളരെ പവർഫുൾ ആണ്. ചേട്ടായിക്ക് ഇനി താഴെ പറയുന്നവയിൽ ഏതെങ്കിലും അറിയണോ?

1. **ArgoCD UI:** ഗ്രാഫിക്കൽ ഇന്റർഫേസ് വഴി ആപ്പുകൾ എങ്ങനെ കാണാം?
2. **Auto-Sync:** ഗിറ്റിൽ പുഷ് ചെയ്താലുടൻ തനിയെ സിങ്ക് ആകുന്നത് എങ്ങനെ സെറ്റ് ചെയ്യാം?
3. **Application YAML:** ഒരു ആപ്പിനെ ArgoCD-യിൽ രജിസ്റ്റർ ചെയ്യുന്ന ഫയൽ എങ്ങനെ എഴുതാം?

ചേട്ടായിയുടെ FastAPI പ്രോജക്റ്റ് ArgoCD-യിലേക്ക് കണക്ട് ചെയ്യാൻ ഞാൻ സഹായിക്കണോ?
