### **1. എന്താണ് Spark & Delta Lake? (ലളിതമായ ഭാഷയിൽ)**

* **Apache Spark**: ഇത് ഒരു സൂപ്പർ ഫാസ്റ്റ് എഞ്ചിൻ പോലെയാണ്. ഒരേസമയം ലക്ഷക്കണക്കിന് ഡാറ്റാ വരികൾ പരിശോധിക്കാൻ ഇതിന് സാധിക്കും.
* **Delta Lake**: ഇത് നമ്മുടെ ഡാറ്റാ സ്റ്റോറേജിന് ഒരു 'ലെയർ' നൽകുന്നു. ഡാറ്റയിൽ എന്തെങ്കിലും തെറ്റ് പറ്റിയാൽ പഴയ വേർഷനിലേക്ക് തിരിച്ചുപോകാനും (Time Travel), ഡാറ്റാ കൃത്യത ഉറപ്പാക്കാനും ഇത് സഹായിക്കും.
* **S3 (Simple Storage Service)**: നമ്മുടെ എല്ലാ ഡാറ്റയും സ്ഥിരമായി സൂക്ഷിച്ചു വെക്കുന്ന AWS-ലെ ഒരു ഡിജിറ്റൽ ലോക്കർ.

---

### **2. ഇൻഫ്രാസ്ട്രക്ചർ റെഡിയാക്കൽ (Terraform)**

ആദ്യം നമ്മുടെ ഡാറ്റ സൂക്ഷിക്കാൻ AWS-ൽ ഒരു **S3 Bucket** ആവശ്യമാണ്. നിങ്ങളുടെ `main.tf` ഫയലിൽ താഴെ പറയുന്ന കോഡ് കൂടി ചേർത്ത് നമുക്ക് ഇത് ക്രിയേറ്റ് ചെയ്യാം.

```hcl
resource "aws_s3_bucket" "food_orders_data" {
  bucket = "alaison-food-orders-delta-lake" # ഇത് യുണീക്ക് ആയിരിക്കണം
  acl    = "private"

  tags = {
    Name        = "Food Orders Data"
    Environment = "Dev"
  }
}

```

### **3. PySpark ഇൻസ്റ്റാൾ ചെയ്യാം**

നിങ്ങളുടെ സെർവറിൽ സ്ഥലം കുറവായതുകൊണ്ട്, വളരെ ശ്രദ്ധയോടെ വേണം ഇൻസ്റ്റാളേഷൻ നടത്താൻ. ആദ്യം `venv` ആക്ടിവേറ്റ് ചെയ്യുക.

