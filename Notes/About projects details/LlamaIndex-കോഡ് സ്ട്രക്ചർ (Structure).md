### LlamaIndex-ന്റെ കോഡ് സ്ട്രക്ചർ വളരെ ലളിതമാണ്. ഇതിനെ പ്രധാനമായും മൂന്ന് ഘട്ടങ്ങളായി തിരിക്കാം: **Data Loading**, **Indexing**, പിന്നെ **Querying**.

താഴെ നൽകിയിരിക്കുന്ന ഉദാഹരണം നിങ്ങളുടെ കയ്യിലുള്ള കുറച്ച് PDF അല്ലെങ്കിൽ ടെക്സ്റ്റ് ഫയലുകളെ എങ്ങനെ ഒരു AI ചാറ്റ് ബോട്ട് ആക്കി മാറ്റാം എന്ന് കാണിച്ചുതരുന്നു.

### LlamaIndex അടിസ്ഥാന ഘടന (Python Code)

```python
import os
from llama_index.core import VectorStoreIndex, SimpleDirectoryReader

# 1. API Key സെറ്റ് ചെയ്യുക (Gemini അല്ലെങ്കിൽ OpenAI ഉപയോഗിക്കാം)
os.environ["GOOGLE_API_KEY"] = "YOUR_GEMINI_API_KEY"

# 2. ഡാറ്റ ലോഡ് ചെയ്യുക (Data Loading)
# 'data' എന്ന ഫോൾഡറിലുള്ള എല്ലാ ഫയലുകളെയും ഇത് വായിക്കും
documents = SimpleDirectoryReader("./data").load_data()

# 3. ഇൻഡക്സ് നിർമ്മിക്കുക (Indexing)
# ഇത് നിങ്ങളുടെ ഡാറ്റയെ AI-ക്ക് മനസ്സിലാകുന്ന രീതിയിലുള്ള (Vectors) രൂപത്തിലേക്ക് മാറ്റുന്നു
index = VectorStoreIndex.from_documents(documents)

# 4. ക്വറി എഞ്ചിൻ തയ്യാറാക്കുക (Querying)
query_engine = index.as_query_engine()

# 5. ചോദ്യം ചോദിക്കുക
response = query_engine.query("ഈ ഡോക്യുമെന്റിലെ പ്രധാന കാര്യങ്ങൾ എന്തൊക്കെയാണ്?")
print(response)

```

### ഈ സ്ട്രക്ചറിന്റെ പ്രവർത്തനം (Architecture)

LlamaIndex എങ്ങനെയാണ് പ്രവർത്തിക്കുന്നത് എന്ന് താഴെ പറയുന്ന ഘട്ടങ്ങളിലൂടെ മനസ്സിലാക്കാം:

1. **Readers (Loading):** നിങ്ങളുടെ ലോക്കൽ ഫോൾഡർ, Notion, Slack, അല്ലെങ്കിൽ ഡാറ്റാബേസ് എന്നിവയിൽ നിന്ന് വിവരങ്ങൾ ശേഖരിക്കുന്നു. ഇതിനെ `SimpleDirectoryReader` എന്ന് വിളിക്കുന്നു.
2. **Documents & Nodes:** വലിയ ഫയലുകളെ ചെറിയ കഷ്ണങ്ങളാക്കി (Chunks) മാറ്റുന്നു. ഇതിനെയാണ് 'Nodes' എന്ന് പറയുന്നത്.
3. **Index (Storage):** ഈ നോഡുകളെ ഒരു വെക്റ്റർ രൂപത്തിൽ സൂക്ഷിക്കുന്നു. ഇത് പിന്നീട് വേഗത്തിൽ തിരയാൻ സഹായിക്കുന്നു.
4. **Query Engine:** നിങ്ങൾ ഒരു ചോദ്യം ചോദിക്കുമ്പോൾ, ഇൻഡക്സിൽ നിന്ന് ഏറ്റവും അനുയോജ്യമായ വിവരങ്ങൾ കണ്ടെത്തി LLM-ന് (Gemini/GPT) നൽകുന്നു.

### ഇതിന്റെ ഗുണങ്ങൾ:

* **കുറഞ്ഞ കോഡ്:** വളരെ കുറഞ്ഞ വരികളിൽ ഒരു RAG സിസ്റ്റം ഉണ്ടാക്കാം.
* **Data Connectors:** 100-ലധികം വ്യത്യസ്ത ഡാറ്റാ സ്രോതസ്സുകളുമായി (Data sources) ഇത് എളുപ്പത്തിൽ ബന്ധിപ്പിക്കാം.
* **Smart Retrieval:** ലാംഗ്ചെയിനിനേക്കാൾ വേഗത്തിൽ കൃത്യമായ ഡാറ്റ കണ്ടെത്താൻ ലാമ ഇൻഡക്സ് സഹായിക്കാറുണ്ട്.

നിങ്ങൾ ഇപ്പോൾ ചെയ്തുകൊണ്ടിരിക്കുന്ന **Pet Cart** പ്രോജക്റ്റിൽ കസ്റ്റമർ സപ്പോർട്ടിനായി ഒരു ചാറ്റ് ബോട്ട് വേണമെന്നുണ്ടെങ്കിൽ, LlamaIndex ഉപയോഗിക്കുന്നത് വളരെ എളുപ്പമായിരിക്കും.
